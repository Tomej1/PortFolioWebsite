Story

Shura's Gauntlet is a fast paced boss rush fighting game, where the player has the freedom to choose the opponent they face in battle. Prove yourselves in battle and you will be able to challenge even stronger opponents.

Project information

  • Genre: Fighting , action
  • Project period: 10 weeks
  • Engine: Unity 3D
  • Group: 10 members

    1. The Fighting mechanics were created for fast paced and interactive combat. In order to make the game more enjoyable different combo sequences were made. By creating lists and dictionaries the program could handle all the players inputs in order to display the specific fighting combos aswell as animations. The following images show how I through code was able to connect specific inputs from multiple devices with certain combo outputs. Since just creating standard attack patterns did not feel engaging enough for the player I choose to also add in combo follow up systems, which matched the players inputs with button inputs in the combo lists in order to execute additional attacks. This makes it so that the combos keeps being performed as long as the player presses the buttons and the combo is a viable combo in the lists, if not the combo is cleared and the player can start new attack patterns.

    Main take aways from the project

    I was responsible for creating a fast paced fighting system for the player and the AI-characters. This project was extremely fun for me to make since i had no prior knowledge when it came to coding fighting mechanics. This required me to make tons of research while doing alot of trial and error. The fighting was created from a playerinputsystem which took information from the players input and checked those inputs through list of fighting inputs in order to match it with the correct action. This made the fighting system in the game responsive and expansive since it allowed for continious implementation of more fighting movesets. I realized during this project that playtesting is one of the most crucial things when creating games. As the fighting system which i created and implemented worked very well in the Unity engine builder it did not quite give the desired result in the build version of the game. This is something I look forward to improving in the future as well as programming and creating fun and interesting systems for future games.
